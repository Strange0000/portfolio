# Sumit Jaiswal - Personal Portfolio 🚀

A modern, highly interactive, and premium personal portfolio website designed to showcase my skills, projects, and journey as a Machine Learning Engineer and Software Developer.

![Portfolio Preview](portfolio_prev.jpeg) <!-- You can replace this with an actual screenshot of the website later -->

## ✨ Features

- **Striking UI/UX**: Built with a sleek glassmorphism design, vibrant gradients, and micro-animations to create a premium feel.
- **Dark & Light Mode**: Fully responsive theme toggling for perfect visibility in any environment.
- **Real-Time Feedback System**: Integrated with **Firebase Firestore** to allow visitors to leave live comments that instantly sync across all browsers.
- **AI Voice Assistant**: An embedded voice synthesizer that reads out sections of the portfolio as you scroll for an accessible, futuristic experience.
- **Interactive Elements**: Magnetic buttons, 3D tilt-effect cards, and cursor-tracking glow effects.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.

## 🛠️ Technologies Used

- **Frontend Core**: HTML5, Vanilla CSS3, Vanilla JavaScript
- **Backend/Database**: Firebase (Firestore) for real-time comment synchronization
- **Icons**: FontAwesome 6
- **Hosting**: GitHub Pages

## 💻 Local Setup (Development)

If you'd like to run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Strange0000/portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```
3. **Open the project:**
   Simply open `index.html` in your web browser, or use an extension like **Live Server** in VS Code for a better development experience.

*Note: The Firebase database credentials are included in `firebase-config.js` to allow real-time testing, but they are scoped to public read/write for demo purposes.*
